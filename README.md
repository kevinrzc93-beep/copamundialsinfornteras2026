# 🏆 Copa Mundial Sin Fronteras - Pitch Deck Web

Un *pitch deck* web interactivo, responsivo y de alto impacto diseñado para presentar la visión de **Copa Mundial Sin Fronteras** a inversores. Este proyecto traduce una presentación de negocios tradicional en una experiencia web fluida y moderna.

## 🚀 Visión General

Mientras el mundo del fútbol se limita a marcadores fríos y estadísticas planas, **Copa Mundial Sin Fronteras** busca documentar la historia, la resiliencia y la conexión global de las comunidades. Este *pitch deck* refleja esa misma innovación a través de un diseño web inmersivo.

## ✨ Características Técnicas y de Diseño

- **Zero Dependencies:** Construido 100% con HTML5 y CSS3 puro. Sin necesidad de instalar React, Vue, Tailwind o Bootstrap.
- **Diseño "Dark Mode" Premium:** Paleta de colores de alto contraste (`#0B0F19` base) para reducir la fatiga visual y resaltar las métricas clave.
- **Glassmorphism UI:** Uso de fondos translúcidos, desenfoques (`backdrop-filter`) y bordes iluminados para un aspecto moderno y tecnológico.
- **100% Responsivo:** Adaptable a dispositivos móviles, tablets y pantallas ultra anchas (Desktop-first adaptado a Mobile).
- **Flujo Cognitivo Optimizado:** Estructurado en un embudo narrativo de 5 fases para guiar la psicología del inversor.

## 📂 Arquitectura de la Presentación

El documento está dividido en 5 secciones estratégicas (*Phases*):

1. **Hook (`#hook`):** El problema raíz y la métrica de impacto (5,000 Millones).
2. **Interest (`#interest`):** Contraste visual de la necesidad del mercado y los puntos de dolor (Pain points).
3. **Desire (`#desire`):** La solución (El Dashboard) y las barreras de entrada (*Moats*).
4. **Action (`#action`):** Fundamentos materiales, embudo de mercado (TAM, SAM, SOM) y Roadmap de ejecución.
5. **Close (`#close`):** Llamado a la acción (CTA) final para el inversor.

## 🛠️ Cómo ejecutar el proyecto

Dado que el proyecto no requiere compiladores ni dependencias, su ejecución es inmediata:

1. Clona o descarga este repositorio.
2. Navega hasta la carpeta del proyecto.
3. Haz doble clic en el archivo `index.html` o ábrelo directamente en cualquier navegador web moderno (Chrome, Safari, Firefox, Edge).

*Opcional: Si usas VS Code, puedes utilizar la extensión "Live Server" para previsualizar los cambios en tiempo real si decides editar el código.*

## 🎨 Paleta de Colores (CSS Variables)

El proyecto utiliza un sistema de variables nativas para facilitar la personalización:

- `--bg-base: #0B0F19` (Fondo principal)
- `--bg-card: #151C2C` (Fondo de componentes)
- `--accent-cyan: #00E5FF` (Acento tecnológico)
- `--accent-yellow: #FFC400` (Resaltado/CTA)
- `--accent-red: #FF3D00` (Puntos de dolor)
